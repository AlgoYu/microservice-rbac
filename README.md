# INTRODUCE
    This is a scaffold for java developer to quick build a project. It based micro-service architecture, use RBAC model to control permission. Integration of common Java development frameworks.
# PREREQUISITES
    1. JDK 1.8+
    2. Maven 3.6.3+
    3. Nacos 1.3.2+
    4. Zipkin 2.21.7+
    4. Sentinel 1.8.0+
